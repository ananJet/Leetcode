# 714. 买卖股票的最佳时机含手续费

两横纵坐标交互型的动态规划

横坐标为日期，纵坐标为0代表买入，1代表卖出，dp记录流水

状态转移方程

在该点买入的最大值等于在前一点买入和在前一点卖出在该点买入两者间取最大

```math
dp[i][0] = Math.max(dp[i - 1][0], dp[i - 1][1] - prices[i]);
```

在该点卖出的最大值等于在前一点卖出和在前一点买入在该点卖出两者间取最大


```math
dp[i][1] = Math.max(dp[i - 1][1], dp[i - 1][0] + prices[i] - fee);
```
