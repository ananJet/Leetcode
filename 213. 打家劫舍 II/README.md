# 213. 打家劫舍 II

循环数组，影响的只是数组的第一个和最后一个，其余数组可看作不循环数组解决子问题，可分为偷第一个和不偷第一个的情况，分别对剩余数组做动态规划（自底向上不递归的动态规划）

状态转移方程

`dp(n) = max{dp[n - 2] + a[n], dp[n - 1]}`

边界条件

1、不偷第一个

`dp(1) = a[1]`

`dp(2) = max{a[1], a[2]}`

`res1 = dp(n)`

2、偷第一个

`dp(2) = a[2]`

`dp(3) = max{a[2], a[3]}`

`res2 = dp(n - 1) + a[0]`
